# Data-analysis-dashboard
Manufacturing data analysis project using Excel, SQL, Power BI, and Tableau to analyze production, quality, and performance metrics.

# Manufacturing Data Analysis Dashboard
##Dataset used


## Project Overview
This project focuses on analyzing manufacturing data to gain insights into production efficiency, quality, and overall performance.

## Tools & Technologies
- Excel (Data cleaning, preprocessing)
- SQL (Data querying and analysis)
- Power BI (Interactive dashboards)
- Tableau (Data visualization and insights)

## Key Analysis Areas
- Manufacture Qty
- Rejected Qty 
- Processed Qty
- Wastage Qty
- Employee Wise Rejected Qty
- Machine Wise Rejected Qty 
- Production Comparison trend 
- Manufacture Vs Rejected
- Department Wise Manufacture Vs Rejected
- Emp Wise Rejected Qty


## Deliverables
- Cleaned and structured datasets
- SQL queries for analysis
- Power BI dashboard
- Tableau dashboard

## Purpose
This project is created for learning and practice purposes to demonstrate end-to-end data analysis skills in a manufacturing domain.
